## Fix 1: Rate Limiter Lock Contention
**File:** `utils/helpers.py`

**Problem:** `acquire()` held `asyncio.Lock()` during `asyncio.sleep()`, serializing all workers. Effective throughput: 3.75 tokens/min.

**Fix:** Sleep outside lock. Lock only held during check-and-decrement (~μs).

```python
# BEFORE
async with self._lock:
    while True:
        if self.tokens >= n:
            self.tokens -= n
            return
        await asyncio.sleep(...)  # blocks all other workers

# AFTER
while True:
    async with self._lock:
        if self.tokens >= n:
            self.tokens -= n
            return
    await asyncio.sleep(...)  # workers run independently
```
